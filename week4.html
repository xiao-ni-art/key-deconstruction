Week 04 / Making Field

第四周：把钥匙研究放回制作现场。

第四周不再只停留在概念网络里，而是回到真实的制作环境：工作台、材料、工具、电脑、打印机和空间秩序。钥匙的“打开”在这里变成从想法到样机的通道。

查看现场分析 进入第五周

Field Observation

第四周的核心不是“想一个产品”，而是观察产品如何被做出来。

01

工作台是入口

桌面看似只是摆放工具的地方，但它决定了设计从哪里开始。干净的空桌面像一个可被开启的空间，任何草图、模型、材料都可以在这里被放大成下一步。

02

材料是钥匙

线材、纸盒、零件、废料和工具不是杂物，它们都是进入不同制作路径的钥匙。材料的硬度、颜色、尺寸和可加工性会反过来决定方案能不能成立。

03

设备是锁芯

3D 打印机像一个把数字模型转为实体的锁芯。模型能否成功输出，取决于喷嘴、平台、切片参数、支撑结构和材料状态是否彼此匹配。

04

失败是反馈

打印失败、材料堆积、桌面混乱并不只是问题，它们暴露了流程里的阻塞点。第四周开始把失败看成设计信息，而不是单纯的错误。

Research Translation

从“钥匙”到“制作系统”

前三周里，钥匙被拆解成结构、语义和产品方向。第四周我开始把它放回真实制作现场：如果钥匙意味着打开，那么工作台就是打开设计过程的第一个场域。

一件产品不是突然出现的，它需要经历资料收集、模型生成、材料选择、切片设置、打印测试、失败修正和最终装配。这个过程本身像一把复杂的钥匙，每一步都要和下一步匹配。

因此第四周的重点是建立“制作链路意识”：不是只展示一个结果，而是展示结果怎样从环境、工具和反复试错中被打开。

环境桌面 / 光线 / 空间
工具电脑 / 打印机 / 辅助工具
材料线材 / 纸盒 / 废料
流程建模 / 切片 / 打印 / 修正

Making Logic

第四周把设计实践拆成五个可汇报的环节。

01清理场地

让桌面从杂乱状态变成可工作的起点，先打开空间。

02确认资源

盘点线材、工具、设备和零件，判断哪些方案可做。

03连接电脑

用电脑完成资料、建模、参数设置和输出准备。

04打印样机

通过 3D 打印把抽象想法转化成可触摸的测试对象。

05修正反馈

把失败样机、废料和问题记录重新输入下一轮设计。

Connection To Key Project

它和“钥匙解域”的关系在哪里？

打开空间

清理工作台让制作可以发生，对应钥匙从封闭到开放的状态转换。

匹配参数

打印机、材料和模型必须匹配，类似钥匙牙花与锁芯之间的对应关系。

接管流程

电脑和设备共同接管想法，把概念导入一条可执行的制造链路。

重构废料

失败和废料被重新理解为下一轮设计的材料，而不是被遗弃的终点。

Week 04 Conclusion

第四周结论

第四周让我意识到，钥匙的“解锁”不只存在于概念里，也存在于制作流程里。一个设计能否被打开，取决于空间是否可用、材料是否合适、设备是否稳定、参数是否匹配，以及失败能否被重新理解。

这周把研究从“想法”推进到“现场”：钥匙不只是产品形象，也是一种组织制作过程的方法。
